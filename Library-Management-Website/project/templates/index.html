Technologies used:

The program utilizes Python3 as the logic to mediate between the database which runs on Sqlite3 and GUI which is written using Jinja2 template engine.
Flask acting as the main framework, and Sqlalchemy as ORM

Usage

Instructions

Idealy, you'd want to start by adding clients and books to your database

Once the previous step is completed you can go ahead and create new logs whenever a book is getting borrowed.
The borrow date would get recorded automatically whenever a new log is created as well as the return date whenever an open log is closed

Notes

The overdue attribute of logs gets updated on startup and at 12am while at runtime.

With that being said, looking up overdued logs directly from the database may not produce accurate results

Searching records is case sensitive. for ease of use, please use the autocomplete suggestions
